# Manual UI Checklist

## Theme Switching

1. Open `DeepSeed.html`.
2. Open Settings panel.
3. Switch theme: Dark -> Ocean -> Forest -> Light -> Dark.
4. Confirm background, panels, and code blocks update each time.
5. Refresh the page and confirm the last selected theme persists.

## Settings Persistence

1. Change parser mode to `diff_v1`.
2. Change markdown mode to `strict`.
3. Refresh the page.
4. Confirm both selections persist.

## Snippet Tags Persistence

1. Edit snippet tags in Settings (add a custom tag).
2. Click Save Tags.
3. Refresh the page.
4. Confirm custom tags are still present and buttons render.

## Base Path Persistence

1. Change Base Path.
2. Click Load.
3. Refresh the page.
4. Confirm Base Path persists.

## Undo Import

1. Import a small change (one file).
2. Click Undo Last Import.
3. Confirm file tree and counts revert.
