# Roadmap

## Milestone 1: Reliability Baseline

- Harden parser edge-case handling.
- Add apply safety checks and clearer failure messages.
- Expand parser tests with malformed and mixed-format responses.

## Milestone 2: UX Productivity

- Improve change previews and conflict visibility.
- Add richer filtering and search over pending/apply states.
- Improve keyboard-driven workflows.

## Milestone 3: Team-Ready Workflows

- Add import/export formats for collaboration.
- Add stable extension points for alternate viewers over parser logic.
- Add contribution quality gates and release process.

## Milestone 4: Optional Integrations

- Optional backend-assisted workflows (feature-flagged).
- Optional MCP/tool-calling compatibility.
- Optional provider abstraction for multiple model APIs.
